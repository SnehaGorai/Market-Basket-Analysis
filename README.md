# Market Basket Analysis System

A Software Engineering project that applies Market Basket Analysis to discover relationships between products frequently purchased together. The project demonstrates the complete software development lifecycle, including requirements analysis, system design, UML modeling, testing, and documentation.

## Features

* Frequent Itemset Generation
* Support Calculation
* Confidence Calculation
* Association Rule Generation
* Buying Pattern Analysis
